# Exemplos de Input para Lead Qualification Agent

## Exemplo 1 - Lead de Alta Qualidade (Score esperado: 8-10)

```
Nome: Carlos Silva
Email: carlos.silva@techcorp.com.br
Empresa: TechCorp Ltda
Cargo: Diretor de TI
Telefone: (11) 98765-4321

Mensagem: Estamos procurando urgentemente uma solução de automação para nosso departamento de vendas. 
Temos um orçamento aprovado de R$ 50.000 e precisamos implementar dentro dos próximos 30 dias. 
Já avaliamos algumas opções no mercado e gostaria de agendar uma demonstração do produto na próxima semana.
```

## Como usar estes exemplos:

1. Copie o conteúdo de um dos exemplos acima
2. Execute o agente de qualificação de leads
3. Cole o texto quando solicitado o input
4. Observe como o agente:
   - Valida a completude das informações
   - Pontua o lead de 1-10
   - Recomenda ações apropriadas
